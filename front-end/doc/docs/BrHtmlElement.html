Example
app(_=> {
    // 사용자 정의 엘리먼트 등록
    app.el.set('header', el=> {
        // home 엘리먼트 생성
        const home= el.a('Home', {href:'/'});
        // nav 엘리먼트 생성
        const nav= el.ul({class:'nav'}, [
            el.li(el.a('Menu1', {href:'/menu1.html', class:'nav-link px-2 text-white'})),
            el.li(el.a('Menu2', {href:'/menu2.html', class:'nav-link px-2 text-white'}))
        ]);

        return el.header({class:'text-bg-dark'}, [
            el.div({class:'flex'}, [
                // home, nav 엘리먼트 추가
                home, nav
            ])
        ]);
    });
});

app(_=> {
    // 사용자 정의 엘리먼트 취득
    const header= app.el.get('header');

    // BrApp.addEl App(body)에 엘리먼트 추가
    app.addEl(app.el.div({class:'wrap'}, [
        // 헤더 추가
        header,
        // 메인 태그 생성 및 추가
        app.el.main({class:'container'})
    ]));
});

set(key, procedure)

Description:

  • BrHtmlElement
    BrHtmlElement.set

    사용자 정의 엘리먼트를 등록 합니다.
    등록된 엘리먼트는 get(BrHtmlElement.get)을 이용하여 사용 합니다.

Example
// BrHtmlElement.get으로 사용 가능 합니다.
app.el.set('myEl', el=> {
    const div= el.div([
        el.p('버튼을 클릭 하세요.'),
        el.button('버튼', button=> {
            button.event('click', e=> alert('버튼을 클릭 했어요'));
        })
    ]);

    return div;
});

tagName(…args)

Description:

  • BrHtmlElement
    BrHtmlElement.tagName

    엘리먼트 객체를 생성합니다.

Example
app.el.div('div 태그생성');

app.el.div('div 클래스 추가', {class:'contents'});

app.el.ul({class:'contents'}, [
    app.el.li('li A'),
    app.el.li('li B'),
]);

app.el.button('클릭 이벤트', button=> {
    button.event('click', e=> alert('1'));
});
Parameters:
Name Type Attributes Description
args any <repeatable>

엘리먼트 생성 옵션 파라미터로 타입별 처리는 다음과 같습니다.

  • string: 엘리먼트의 TEXT를 처리 합니다.
  • object: 엘리먼트의 어트리뷰트를 세팅 합니다.
  • array: 엘리먼트의 자식 노드를 구성 합니다.
  • function: (tag: BrHtmlElementItem)=> void
    엘리먼트 생성시 호출 되는 함수로 생성된 엘리먼트 객체를 제공합니다.
