BrHtml
BrHtml.BrHtmlVo

data-vo 지시자로 선언된 vo핸들러 객체 입니다.

Type:
  • object
Example
// html
// <section id="header">
//     <h1 data-vo="title">form</h1>
//     <input data-vo="input" type="text">
//     <p data-vo="output"></p>
// </section>

// vo 핸들러 취득
const vo= app.html.dataVo('#header');

// value set/get
vo.setValues({input:'1'});
console.log(vo.getValues());

// text set/get
vo.setTexts({title:'form title'});
console.log(vo.getTexts());

// html set/get
vo.setHtmls({output:'<strong>outHtml</strong>'});
console.log(vo.getHtmls());

Description:

    BrHtml.insert

    어플리케이션 컨텐츠(BODY)에 엘리먼트 노드를 추가 하며, 추가 된 노드 객체를 반환 합니다.

Example

// main앞에 head 노드를 추가 합니다.

// 배경색 변경
