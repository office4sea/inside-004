await(key, dataopt) → {Promise.<any>}

Description:

  • BrAsync
    BrAsync.await

    비동기 서비스를 활성화 하며, 다른 프로세스에서 처리 완료까지 대기 합니다.
    BrAsync.resolve(비동기 귀결), BrAsync.reject(비동기 거부)를 통해 비동기 서비스를 완료 합니다.

Example
// 비동기 서비스 호출, 프로세스 완료 까지 대기 합니다.
// BrAsync.suspend에 등록한 경우 해당 키값의 함수를 실행 합니다.
// 정상완료: then 으로 결과값 반환 합니다.
// 예외완료: catch 로 예외값 반환 합니다.
app.async.await('coMessage', {a:1})
    .then(result=> {
        // result: BrAsync.resolve에서 전달한 인자값 {b:1}
        app.debug.success('coMessage', '비동기 처리 완료', result);
    })
    .catch(reason=> {
        // reason: BrAsync.reject에서 전달한 인자값 {c:1}
        app.debug.danger('coMessage', '비동기 처리 거부', reason);
    })
    .finally(_=> {
        app.debug.info('coMessage', '비동기 완료');
    });

// async / await 키워드를 사용한 축약법
async function fetchMessage(payload) {
    const result= await app.async.await('coMessage', payload);
    app.debug.success('coMessage', '비동기 처리 완료', result);
    return result;
}
